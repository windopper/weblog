입력 텐서를 반복시켜서 리턴해주는 메서드이다.

```python
tf.tile(
    input, multiples, name=None
)
```


# 예제

```python
x = tf.constant([[1, 2, 3], [4, 5, 6]], dtype=tf.int32)
y = tf.constant([3, 3], dtype=tf.int32)
tf.tile(x, y)
```
