# 개요

DFS와 BFS는 그래프를 탐색하는 데 사용되는 알고리즘입니다. DFS는 깊이 우선 탐색이라고하며, 그래프의 깊은 부분을 우선적으로 탐색합니다. BFS는 너비 우선 탐색이라고하며, 그래프의 가까운 부분부터 탐색합니다.

# 전략

## 배열에 접근여부를 설정할 때 차원의 크기가 너무 크다면

일반적으로 `visited` 또는 `check` 라는 이름을 가진 배열을 통해 해당 좌표에 접근했는지를 저장합니다.

그러나 입력 값이 너무 크다면 메모리 초과가 뜨기 쉬우므로 이를 간단하게 채우는 방법이 있습니다.

입력 값끼리의 상관관계가 있다면 이를 이용하여 차원의 크기를 줄일 수 있습니다.
