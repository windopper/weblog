# 개요

세그먼트 트리의 노드 부분이 배열 인 것 빼고는 다른 점이 없다.

합치기 연산 시에 c++의 내장 함수 merge()를 사용하면 굉장히 간편하게 구현할 수 있다.

일반적으로 머지 소트 트리의 질의는 특정 구간에 있는 원소의 위치를 찾는 것이다.
