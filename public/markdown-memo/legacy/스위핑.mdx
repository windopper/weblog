# 개요

스위핑(Sweeping) 알고리즘은 평면상에서 여러 개의 선분이나 점들이 주어졌을 때, 그들이 교차하는 지점의 좌표 등을 찾는 알고리즘입니다. 스위핑 알고리즘은 대개 빠른 시간복잡도와 함께 많은 문제에서 활용됩니다.
