# 개요


## 다익스트라(Dijkstra) 알고리즘이란?

다익스트라 알고리즘은 그래프에서 최단 경로를 찾는 알고리즘 중 하나입니다. 음의 가중치를 갖는 간선을 포함하지 않는 경우에 이용할 수 있습니다.

## 알고리즘 개념

1. 시작 정점을 기준으로 거리 값을 할당하고, 시작 정점 외의 다른 정점들은 무한대의 값으로 초기화합니다.
1. 방문하지 않은 정점 중에서 가장 짧은 거리의 정점을 선택합니다.
1. 해당 정점에서 인접한 정점들의 거리 값을 갱신합니다.
1. 위 과정에서 2, 3번을 반복합니다.
