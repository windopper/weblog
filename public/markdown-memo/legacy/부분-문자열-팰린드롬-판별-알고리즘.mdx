# 개요

팰린드롬은 어떤 문자열이 양쪽에 대해 대칭하는 것을 의미합니다.

대표적으로 `abccba` 의 경우가 있으며 `aba` 와 `a` 의 경우도 성립합니다.


# Naive한 접근

어떤 문자열이 팰린드롬인 경우를 판별하기 위한 가장 단순한 방법은 해당 문자열을 뒤집었을 떄 같은 문자열이 나오는 방법입니다.

그러나 해당 알고리즘을 부분 문자열에 대해 가장 긴 팰린드롬 길이를 구할 떄 사용한다면 $O(N^3)$의 시간복잡도가 나오게 됩니다.

> *모든 경우의 수를 계산하는데  **$N^2$**, 팰린드롬을 판별하는데 **$N$**. 따라서 **$O(N^2\times N)$*


# DP를 활용한 $O(N^2) $ 풀이

```c++
// 자기 자신은 항상 팰린드롬
for(int i=1; i<=N; i++) dp[i][i] = 1;

// i, i+1번째 값이 같다면 팰린드롬
for(int i=1; i<=N-1; i++) if(arr[i] == arr[i+1]) dp[i][i+1] = 1;

// 1-3, 2-4, ...번째 값이 같다면 2-2도 팰린드롬인지 확인 한 후 맞다면 팰린드롬
// l-r 번쨰 값이 같다면 l-1 - r+1도 팰린드롬인지 확인 한 후 맞다면 팰린드롬
for(int i=2; i<=N-1; i++) {
	for(int l=1; l<=N-i; l++) {
	  int r = l + i;
		if(arr[l] == arr[r] && dp[l+1][r-1]) dp[l][r] = 1;
  }
}
```
