# 개요

트라이(Trie)는 문자열 검색을 위한 트리 자료구조입니다. 이 자료구조는 문자열에서 일치하는 문자열을 빠르게 찾을 수 있도록 해줍니다. 트라이는 각 노드가 문자 하나를 나타내는 트리입니다. 루트 노드에서부터 각 문자열의 글자를 따라 내려가며, 각 노드에서 해당 글자를 나타내는 자식 노드로 이동합니다. 이를 통해 문자열을 검색할 수 있습니다.
